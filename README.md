# Networking Portfolio – Fatoumata Y Bocoum

This repository contains my personal networking portfolio website, built with a single HTML file that includes HTML, CSS, and a small amount of JavaScript.

The site is hosted with GitHub Pages at:

- Live site: https://fbocoum1.github.io/Networking-portfolio/
- Repository: https://github.com/fbocoum1/Networking-portfolio

## Sections

- **Home** – Introduction, profile photo, short biography, and quick links to my resume, biography PDF, and GitHub repository.
- **Skills & Knowledge** – Key networking, tools, scripting, and professional skills presented as quick tags.
- **Hands-On Labs** – Short writeups of labs such as Active Directory, IPv4 subnetting, and VLANs & trunking.
- **Contact** – Contact information and links to my GitHub and (optionally) LinkedIn profile.

## Files

- `index.html` – Main single-page portfolio (HTML, CSS, and JavaScript).
- `Fatoumata_Y_Bocoum-Resume.pdf` – My current resume.
- `My-Biography.pdf` – A one-page biography with more background detail.
- `images/` – Optional screenshots and profile photo used in the portfolio.
