Skills I’m Learning and Practicing

Networking & Troubleshooting

  • Designing simple IP addressing plans: create IPv4 subnets for small networks, assign host addresses, and choose gateways so devices can communicate.
  • Configuring VLANs and trunk ports in Packet Tracer: separate traffic for different groups (for example students vs. staff) and verify connectivity.
  • Basic router and switch configuration: set hostnames, passwords, interfaces, and save running configurations in Packet Tracer labs.
  • Using ping and tracert for troubleshooting: test connectivity, identify where traffic stops, and decide if the problem is addressing, cabling, or configuration.

Windows & Active Directory

  • Installing Windows Server in a virtual machine: create a VM, configure network settings, and prepare it to become a domain controller.
  • Creating a basic Active Directory domain: promote a server to domain controller, create OUs, users, and groups, and join Windows clients to the domain.
  • Managing user accounts and simple policies: reset passwords, disable accounts, and apply basic group policies such as password or desktop restrictions.

Tools, Scripting & Web

Hands-On Labs

  • Verified inter-VLAN communication using router-on-a-stick.
